Anteojos para Asistencia de Personas con Discapacidad Visual

Introducción

Este proyecto consiste en el desarrollo de unos anteojos inteligentes diseñados para brindar apoyo a personas con discapacidad visual mediante la detección de obstáculos cercanos. El sistema utiliza un Arduino Nano como unidad de control, sensores ultrasónicos para medir distancias y actuadores que alertan al usuario cuando un objeto se encuentra dentro de un rango determinado.

El propósito del proyecto es aumentar la seguridad y la autonomía de las personas con discapacidad visual durante sus desplazamientos, permitiéndoles identificar obstáculos que podrían representar un riesgo.

Planteamiento del Problema

Las personas con discapacidad visual enfrentan diversos desafíos al desplazarse de manera independiente. Aunque existen herramientas como el bastón blanco y los perros guía, algunos obstáculos ubicados a la altura de la cabeza o del torso pueden ser difíciles de detectar, aumentando el riesgo de accidentes. Por ello, surge la necesidad de desarrollar dispositivos tecnológicos complementarios que ayuden a detectar obstáculos de forma anticipada y proporcionen alertas inmediatas al usuario.

Justificación

La implementación de tecnología asistiva puede mejorar significativamente la calidad de vida de las personas con discapacidad visual. Este proyecto propone una solución de bajo costo utilizando componentes electrónicos accesibles y fáciles de programar.

Los anteojos inteligentes permiten detectar objetos cercanos y generar alertas mediante sonidos o vibraciones, proporcionando información adicional sobre el entorno y contribuyendo a una movilidad más segura.

Objetivo General

Diseñar e implementar un sistema de anteojos inteligentes basado en Arduino Nano para la detección de obstáculos y asistencia en la movilidad de personas con discapacidad visual.

Objetivos Específicos

  • Detectar obstáculos mediante sensores ultrasónicos.
  • Procesar la información utilizando un Arduino Nano.
  • Alertar al usuario mediante buzzers o motores vibradores.
  • Evaluar diferentes configuraciones de sensores y actuadores.
  • Mejorar la seguridad durante el desplazamiento.

Funcionamiento del Sistema

El sistema utiliza sensores ultrasónicos para medir la distancia entre el usuario y los objetos cercanos. Los sensores emiten ondas ultrasónicas que rebotan en los obstáculos y regresan al sensor en forma de eco.

El Arduino Nano procesa esta información y calcula la distancia detectada. Cuando un objeto se encuentra dentro de un rango de seguridad previamente definido, el sistema activa los actuadores correspondientes.Dependiendo de la variante implementada, la advertencia se realiza mediante buzzers que generan sonidos o mediante motores vibradores que producen señales táctiles.

Mientras más cerca se encuentre el obstáculo, la frecuencia o intensidad de la alerta puede incrementarse, permitiendo al usuario reaccionar con mayor rapidez.

Componentes Utilizados

  • Batería portátil o fuente de alimentación
  • Montura de anteojos

Variantes del Proyecto

Variante 1: Dos Sensores con Buzzers

Esta configuración emplea dos sensores ultrasónicos colocados en diferentes posiciones para ampliar el área de detección. Las alertas se generan mediante buzzers.

Características:

  • Cobertura de detección más amplia.
  • Mayor capacidad para identificar obstáculos en diferentes direcciones.
  • Avisos sonoros inmediatos.

    • Ventajas:

  • Mejor detección del entorno.
  • Fácil implementación.
  • Respuesta rápida.

    Desventajas:

  • Puede resultar difícil de utilizar en ambientes muy ruidosos.
  • Dependencia de señales auditivas.

    • Variante 2: Dos Sensores con Vibradores.

    Esta variante utiliza dos sensores ultrasónicos y motores vibradores para emitir alertas táctiles.

    Características:

  • Cobertura amplia.
  • Señales mediante vibración.
  • Mayor discreción durante el uso.

    • Ventajas:

  • No depende del sentido del oído.
  • Funciona adecuadamente en ambientes ruidosos.
  • Retroalimentación táctil constante.

    • Desventajas:

  • Requiere adaptación para interpretar los patrones de vibración.
  • Mayor complejidad de montaje.

    • Variante 3: Un Sensor con Buzzers

    Esta configuración utiliza un solo sensor ultrasónico y buzzers para alertar sobre obstáculos.

    Características:

  • Diseño sencillo.
  • Menor cantidad de componentes.
  • Menor consumo energético.

    • Ventajas:

  • Bajo costo.
  • Fácil mantenimiento.
  • Programación simple.

    • Desventajas:

  • Menor cobertura de detección.
  • Menor precisión para determinar la ubicación del obstáculo.

    • Variante 4: Un Sensor con Vibradores

    Esta versión emplea un sensor ultrasónico y motores vibradores para proporcionar alertas táctiles.

    Características:

  • Diseño compacto.
  • Sistema económico.
  • Fácil integración en la montura.

    • Ventajas:

  • Alertas discretas.
  • Bajo consumo energético.
  • Mayor comodidad para algunos usuarios.

    • Desventajas:

  • Cobertura limitada.
  • Menor capacidad para detectar obstáculos en múltiples direcciones.
